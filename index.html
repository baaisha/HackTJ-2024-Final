Basic statistics terms, Dotpots and Bar Graphs

Note 0.1 — Statistics is the science of learning from data. Data is always collected in real-world context.
Definition 0.2 — A variable is a characteristic of something that can be assigned a number or category.
Example 0.3. —Is the number 2.6 considered a variable? What is an example of one?

Walkthrough.

  1. Can it be assigned a number or meaning?
  2. What is something that you can measure about youself?

Definition 0.4 — The thing in which the category or number is assigned is called the observational unit. It is the objects or people that data is being collected from |observational unit| to |observational unit|; statistics would not be a thing if there was no variability
Definition 0.5 —
  • Quantitative variable measures a numerical characteristic
  • Categorical variable records a group designation; binary variables are categorical variables with only two possible categories
  • A research question often looks for patterns in different variables and looks for relationships between variables
    •
    Definition 0.6 — Dotplot: graphs a quantitative variable. Each quantitative data value is shown as a dot above its location on a number line
    1. Draw a horizontal number line with the variable name
    2. Scale the axis by looking at the minimum and maximum values
    3. Mark a dot above the location of horizontal axis for each observational unit
    Definition 0.7 — Bar graph: used to display data from a categorical variable or compare the size of different quantities (bars are not touching)
    1. Label both axes (categorical variable on the x-axis and quantitative data value on the y-axis)
    2. Draw bars that correspond to each variable and leave room in between each bar
    Note 0.8 — Dotplots and bar graphs are most illuminating when comparing the distribution of a variable between two or more groups
    Definition 0.9 —
    • Tendency: refers to the center of the distribution and where the data points are likely to fall.
    • Consistency: refers to the spread/range of the distribution (synonymous with spread, variability) and how likely the data points are to fall within a certain range
    • When comparing tendency, do not use the word greater or less than, simply compare the values
    Check your Understanding 1.0 —
    1. What does each dot in a dot plot represent?
    2. What type of variable is displayed in dot plots? What type of variable is displayed in bar graphs?
    3. When are dot plots and bar graphs most illuminating?
    4. What is consistency and tendency? How are they different?
